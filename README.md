# 注意事项
- macOS提示文件损坏请执行`sudo xattr -d com.apple.quarantine Fine.app`命令后重新打开；
- 微信v4暂无法反编译，小程序路径需选择包含目录结构如：`...\选择该目录\wx...\12\__APP__.wxapkg`，反编译仅提取了文件做敏感信息查找或者简单可阅读，微信开发者工具无法运行；
- 数据表格默认均可单元格和多范围选中按键复制。
